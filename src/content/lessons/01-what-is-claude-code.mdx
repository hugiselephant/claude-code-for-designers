# What is Claude Code and Why It Changes Everything

Welcome to Module 1! If you're a designer reading this, you're about to discover something
that genuinely changes the game.

## The Problem We've All Felt

You've had the idea. You've designed the mockups. You've even prototyped in Figma.
But then comes the wall — actually *building* it. Turning pixels into a real, working
product that lives on the internet.

Until now, your options were:
- Learn to code from scratch (months or years)
- Hire a developer (expensive, communication overhead)
- Use a no-code tool (limited, locked into their platform)

## Enter Claude Code

Claude Code is an AI development partner that runs in your terminal. You describe what
you want in plain English, and it writes the code for you. But here's what makes it
different from other AI tools:

- **It sees your entire project** — not just a single file
- **It understands context** — your tech stack, your patterns, your conventions
- **It iterates with you** — you review, give feedback, and refine together
- **It explains what it does** — so you actually learn along the way

## Why This Matters for Designers

You already think in systems. You understand user flows, component hierarchies, and
visual patterns. That's 80% of what makes a good developer. Claude Code bridges the
remaining 20% — the syntax, the tooling, the deployment pipeline.

> The best products are built by people who understand both design and engineering.
> Claude Code lets you be that person.

## What We'll Build in This Course

By the end of this 6-week course, you'll have built and deployed a real web application.
Not a tutorial project — *your own idea*, live on the internet. Along the way, you'll learn:

1. How to set up a professional development environment
2. How to communicate with Claude Code effectively
3. Git, GitHub, and version control
4. React, Next.js, and Tailwind CSS
5. Databases and data persistence
6. Deployment and going live
